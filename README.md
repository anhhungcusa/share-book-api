# Share book api

## Requirements

- Node
- MongoDB server(local or cloud)

## Using

Restful server with

 - Express
 - Mongoose + MongoDB(MongoDB Atlas)
 - Authentication with **jwt**


## Install package

Use the package manager [npm](https://www.npmjs.com/) to install dependencies.

``` cmd
npm i
```

## Run local

run with node

``` cmd
npm start
```

run with nodemon

``` cmd
npm run dev
```
